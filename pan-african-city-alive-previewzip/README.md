# Pan African City Alive Preview Storefront

Preview storefront for Pan African City Alive, built as a Vite + React static site.

## What Is Included

- Animated click-the-mask entrance with optional licensed drum ambience.
- Visible Pan-African flag/Africa symbol logo, kente side rails, full hero, founder story, shop categories, product grid, gallery, events, visit, and footer.
- Working local cart, quantity controls, product detail modal, checkout/request form, event RSVP modal, category filtering, search, and newsletter/contact form UI.
- Project-local image library in `public/images/final` and `public/images/client`, derived from supplied/generated assets plus public-source supported reference material.
- Factual business claims are limited to the supplied research PDF and public-source supported items. Inventory, sample prices, and event dates are marked as preview/demo data until confirmed.
- Drum ambience source: "Village Drums of Freedom - Black Africa (djembe mix)" by Gerald Achee, CC BY-SA 3.0, via Wikimedia Commons.

## Local Commands

```bash
npm install
npm run typecheck
npm run build
npm run dev
```

## Deployment

The site is optimized for static deployment on Vercel. Production build output is generated with:

```bash
npm run build
```
